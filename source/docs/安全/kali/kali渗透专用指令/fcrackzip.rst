zip包的爆破

选项:

-D ‘字典’
  字典破解方式

-b
  暴力破解方式
-c
  暴力破解的字符类型，1=1~9,a=a~z,A=A~Z,!=字符,:=所有字符
-v
  啰嗦模式，显示实时爆破信息
-l
  指定破解的密码为几位数
-u
  指定爆破文件
-p
  破解的起始位置
