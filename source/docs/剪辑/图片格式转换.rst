=========================
图片格式转换
=========================



方案一: 使用 convert 命令
=========================


使用 convert 命令, mac需要安装imagemagick (内置于此包命令)::

	brew install imagemagick

缺点: 质量不高

方案二: sips
=========================

mac自带, 质量较高

不过不支持转换为 svg ...

:参考::
	https://blog.csdn.net/qq_41437512/article/details/122619375
