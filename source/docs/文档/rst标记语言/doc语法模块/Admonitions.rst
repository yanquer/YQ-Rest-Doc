=================================
Admonitions
=================================

| 提示信息

Specific Admonitions
=================================

支持以下类型:

- attention
- caution
- danger
- error
- hint
- important
- note
- tip
- warning

如使用tip, 源码::

  .. tip::
    this is a tip msg

**效果**

.. tip::
  this is a tip msg

Generic Admonition
=================================

与上区别更像是支持自定义警告标题

支持的类型:

- admonition

举例, 源码::

  .. admonition:: And, by the way...

    You can make up your own admonition too.

**效果**

.. admonition:: And, by the way...

  You can make up your own admonition too.
