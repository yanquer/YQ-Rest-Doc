虽然可以通过在 conf.py 配置::

  extensions = [
    'sphinx.ext.autodoc',
    'sphinx.ext.autosectionlabel', 	# 允许直接ref使用标题引用(否则需要给标题定义别名),
                                    # 支持跨文档标题引用(注意标题在多个文档中保持唯一),
  ]

主要是 ``sphinx.ext.autosectionlabel`` 来解决, 但是不建议这么做, 因为保不齐什么时候有个标题就是得重名.
