跨文档超链接(引用)
==================================

在一个rst文档中, 小节会自动加入到本文档的超链接中.

而要跨rst文档饮用另一个rst文本的小节/链接, 只有使用::

  :ref:`xxx`

而非::

  xxx_

查看当前已生成的标记

查看当前已生成的ref标记, 一般都是生成在 build/html/objects.inv::

  python -m sphinx.ext.intersphinx build/html/objects.inv
