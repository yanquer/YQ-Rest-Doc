visdom
===================================

启动 visdom server::

	python -m visdom.server

这一步会有一个下载的过程，会比较慢

使用举例::

	import visdom
	import numpy as np

	vis = visdom.Visdom()
	vis.text('he')
	vis.image(np.ones((3, 10, 10)))

tensorboardx
===================================

使用, 先生成一个示例::

	from tensorboardX import SummaryWriter

	writer = SummaryWriter('log')

	for i in range(100):
		writer.add_scalar('a', i, global_step=i)
		writer.add_scalar('b', i**2, global_step=i)
	writer.close()

启动::

	cd log
	tensorboard --log-dir ./
