前言
==========================

将工具版本更新到 `PyCharm2022.3` 后, 在文件或右下角找不到同步设置的地方了, 一度以为是这个功能被砍了, 最后在设置里面找到了

设置里面新增了一个 `设置同步` 选项, 默认禁用, 打开即可.
