==============================
枚举类型值的问题
==============================


.. post:: 2024-03-09 18:21:01
  :tags: c, 问题
  :category: 后端
  :author: YanQue
  :location: CD
  :language: zh-cn


定义一个枚举::

  enum LANGUAGE_TYPE {
      EN = 0,
      ZH_CH,
  };

如果没有赋值::

  enum LANGUAGE_TYPE current_language;

那么默认值就是第一个 `EN`
