对DDD的理解
  最开始的MVC只是抽象出一个Model实体, 做ORM;

  业务复杂度堆积在Service层,
  将Service的业务实现抽象为一个通用的 "领域对象"(可以理解为一个特殊的Model), 就叫DDD
