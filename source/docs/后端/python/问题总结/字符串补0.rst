==================================
字符串补0
==================================


.. post:: 2024-03-07 22:25:04
  :tags: python, 问题总结
  :category: 后端
  :author: YanQue
  :location: CD
  :language: zh-cn


左对齐, 右侧补0::

  In [2]: "123".ljust(5, "0")
  Out[2]: '12300'

右对齐, 左侧补0::

  In [3]: "123".rjust(5, "0")
  Out[3]: '00123'

或者::

  In [4]: "123".zfill(5)
  Out[4]: '00123'

或者使用%::

  In [5]: "%05d" % 123
  Out[5]: '00123'
