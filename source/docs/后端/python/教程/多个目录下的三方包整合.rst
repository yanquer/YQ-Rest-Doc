多个目录下的三方包整合
===========================


.. post:: 2023-02-20 22:06:49

有以下方式

- 将路径直接添加到sys.path
- 将路径加入到环境变量 PYTHONPATH
- 在当前三方包路径下创建一个 ``.pth`` 后缀的文件, 然后将路径按行写入

详见: :ref:`将文件夹加入到sys.path`
