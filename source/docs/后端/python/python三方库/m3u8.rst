m3u8

pypi官网地址: https://pypi.org/project/m3u8/

安装::

  pip install m3u8

媒体文件转换为m3u8格式.
一般与 :doc:`/docs/后端/python/python三方库/ffmpeg` 一起使用.

加载m3u8文件, 参数为本地位置或网络url::

  m3u8.load(file_or_url)
