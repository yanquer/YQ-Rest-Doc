min

min() 函数返回值最小的项目，或 iterable 中值最小的项目。
如果值是字符串，则按字母顺序进行比较。

.. function:: min(iterable, key)

  key用于当 iterable 中元素也为 iterable 时, 定义比较的键/元素

.. function:\: min(n1, n2, n3, ...)
