生成一个字典

支持将两个值的迭代转换为字典::

  In [1]: a = [(1, 2), (2, 3), (3, 4)]

  In [2]: dict(a)
  Out[2]: {1: 2, 2: 3, 3: 4}

  In [3]: dict(tuple(a))
  Out[3]: {1: 2, 2: 3, 3: 4}

  In [4]:

合并多个字典
================================

这里要注意 \*\*参数关键字必须是字符串::

  dict(dict1, **dict2, **dict3)
