JS对象与字符串转换
==============================

ENCODEJSON
  将JS对象转换为字符串,
  例::

    data: {
        envStr: '${ENCODEJSON(envObj)}',
    },
DECODEJSON
  将字符串转换为JS对象,
  例::

    data: {
        envObj: '${DECODEJSON(envStr)}',
    },
