===========================
数据域的向上更新
===========================

重点!

**适用于当前数据链**

主要对前面的不当做一个纠正, 这部分纠正有空再找出来改,

同一个页面的数据域, 向上向下的更新都是支持的,
.. 不过复杂情况下, 默认仅能保持两层, 可以使用 ``trackExpression`` 来确定跟踪的字段;

不同页面的, 比如弹出一个dialog, 相当于是一个新的页面, data 应当是被拷贝了一份,
dialog数据的变化无法影响到上一层

**dialog父级组件的数据可以向下传递**, 但是dialog父级组件的兄弟组件不行.
