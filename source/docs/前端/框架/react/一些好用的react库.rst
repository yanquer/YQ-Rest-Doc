react-tooltip
===============================

安装::

  npm i react-tooltip

用于 **鼠标移入** 时候的提示显示::

  <div>
      <a data-tooltip-id="my-tooltip" data-tooltip-content="Hello world!">
          ◕‿‿◕
      </a>
      <Tooltip id='my-tooltip'></Tooltip>
  </div>
