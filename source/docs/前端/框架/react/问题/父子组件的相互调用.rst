子组件调用父组件的方法
===============================

这个比较简单, 父组件调用子组件的时候, 像普通传参一样,
将需要调用的方法传给子组件即可.

父组件调用子组件的方法
===============================

父组件使用::

  render(){
    return <Child
      ref={view => {this._childView = view || undefined}}
    />
  }

然后通过 ``this._childView`` 调用子组件即可

详情参考: :doc:`/docs/前端/框架/react/hooks/createRef`
