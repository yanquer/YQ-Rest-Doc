=======================================
关于React使用自定义组件却需要使用状态
=======================================

找了很多种方法，
最终只有将需要一种方法: ``将需要用到状态的这部分内容， 分离出去`` ，
至于如何分离， 大致有两种方式

- 函数组件+HOOP
- 继承 ``React.Component`` 的类组件， 使用其中的 `state` 与 `setState`

别无它法， 除非你在自己的这个自定义类中将 ``React.Component`` 中的相关内容重写一遍，
很麻烦
