设置元素透明度

opacity 属性控制元素的不透明度级别，
其值范围从 0 到 1，

- 0 表示完全透明（不可见），
- 1 表示完全不透明（完全可见）。

将元素设置为半透明（透明度为 0.5）::

  .element {
    opacity: 0.5;
  }

.. note::

  有时候不设置背景, 背景自动就是透明的
