==============================
mklink
==============================

创建符号链接。

使用::

  MKLINK [[/D] | [/H] | [/J]] Link Target

          /D      创建目录符号链接。默认为文件符号链接。
          /H      创建硬链接而非符号链接。
          /J      创建目录联接。
          Link    指定新的符号链接名称。 其实应该叫target， 链接到的地址
          Target  指定新链接引用的路径。 其实应该叫source， 真实源地址
                  (相对或绝对)。
