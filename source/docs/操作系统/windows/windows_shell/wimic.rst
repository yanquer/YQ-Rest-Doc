windows下的一个交互式的命令行环境(类似bash shell之类)

帮助用法如下：


命令行帮助

+-----------------+------------+------------------------------+
|    命令         |    例子    |       说明                   |
+=================+============+==============================+
|  /? 或 -?       |            | 显示所有全局开关和别名的语法 |
+-----------------+------------+------------------------------+
| /? class /?     |            | 显示某个命令的信息           |
+-----------------+------------+------------------------------+
| /? memcache /?  |            | 显示某个别名的信息           |
+-----------------+------------+------------------------------+
| /? temperature  |            | 显示别名与动词组合的信息     |
| get /?          |            |                              |
+-----------------+------------+------------------------------+
| /?:Full irq     |            | 显示动词的帮助信息           |
| get /?:Full     |            |                              |
+-----------------+------------+------------------------------+

常用的

查看所有进程::

  wimic process
