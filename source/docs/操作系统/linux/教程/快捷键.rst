==============================
快捷键
==============================

ctrl 组合键
==============================

- ctrl + c    终止目前的命令
- ctrl + d    输入结束（EOF），例如邮件结束的时候
- ctrl + M    enter
- ctrl + s    暂停屏幕输出
- ctrl + q    恢复屏幕输出
- ctrl + u    在提示字符下，将整列命令删除
- ctrl + z    暂停 目前的命令

当输入一串指令时，发现前面写的一串数据是错的，
想要删除游标所在处到最前面的指令内容，应该如何处理::

  ctrl + u
