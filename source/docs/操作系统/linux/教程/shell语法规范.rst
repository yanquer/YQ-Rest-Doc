case语句
=========================

语法如下::

  case 值 in
      模式1)
          命令序列1
          ;;
      模式2)
          命令序列2
          ;;
      ...
      *)
          默认命令序列
          ;;
  esac

- `值` 是需要匹配的表达式或变量。
- `模式` 是用来匹配值的模式，可以使用通配符。
- `命令序列` 是当匹配到相应模式时要执行的一组命令。
- ``;;`` 表示一个模式块的结束，类似于break语句，用于跳出case语句;
  如果不加;;，Shell会继续往下执行后续模式的命令序列，而不进行匹配
- ``*)`` 是可选的默认模式，如果没有匹配到任何模式，将执行对应的命令序列;
  默认模式通常放在最后，用于处理未匹配到任何模式的情况
