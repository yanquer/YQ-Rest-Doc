进程暂停与挂起
==================================

在macos上验证

ctrl + z 暂停进程

恢复

- jobs 查看有哪些进程被暂停
- bg %N 使第N个任务在后台运行（%前有空格）
- fg %N 使第N个任务在前台运行
- jg %n 挂起被暂停的进程，n为进程序号

.. note::

  默认bg，fg不带%N时表示对最后一个进程操作！
