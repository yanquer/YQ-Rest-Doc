gnome-terminal

主要适用于gnome的图形化桌面系统, 用于打开一个新的终端


打开一个终端并执行ls::

  gnome-terminal -- ls

打开一个终端并执行ls, 执行后不退出::

  gnome-terminal -- bash -c 'ls ; bash'
