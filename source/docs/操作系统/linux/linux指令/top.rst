.. 实时动态地查看系统的整体运行情况

显示或管理执行中的程序，实时动态地查看系统的整体运行情况，是一个综合了多方信息监测系统性能和运行信息的实用工具。


选项

-b
  以批处理模式操作；
-c
  显示完整的治命令；
-d
  屏幕刷新间隔时间；
-I
  忽略失效过程；
-s
  保密模式；
-S
  累积模式；
-i<时间>
  设置间隔时间；
-u<用户名>
  指定用户名；
-p<进程号>
  指定进程；
-n<次数>
  循环显示的次数。
