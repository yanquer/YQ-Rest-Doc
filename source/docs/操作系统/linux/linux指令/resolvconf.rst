=====================================
resolvconf
=====================================

.. note::

  这是一个软件包, 不确定有没有指令

与 `/etc/resolv.conf` 相关, 配置域名解析服务地址

如果 `resolvconf` 软件包没有安装， `/etc/resolv.conf` 是一个静态文件。
如果安装了，它是一个符号链接.
此外，它包含有解析策略的初始化信息。如 DNS 是 IP=" `192.168.11.1` ",则包含如下::

  nameserver 192.168.11.1

`resolvconf` 软件包使这个 `/etc/resolv.conf` 文件成为一个符号链接，
并通过钩子脚本自动管理其内容。
