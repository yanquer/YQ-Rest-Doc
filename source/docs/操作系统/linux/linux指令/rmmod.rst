参数：

-f
  强制将该模组移除掉，不论是否正被使用；
-w
  若该模组正被使用，则 rmmod 会等待该模组被使用完毕后，才移除他！

内核模块可见: :doc:`/docs/操作系统/linux/系统服务/内核模块`
