========================
wc
========================

利用wc指令我们可以计算文件的Byte数、字数、或是列数，若不指定文件名称、或是所给予的文件名为"-"，则wc指令会从标准输入设备读取数据。


语法::

	wc [-clw][--help][--version][文件...]

参数
========================

-c, --bytes, --chars  只显示Bytes数。
-l, --lines           显示行数。
-w, --words           只显示字数。一个字被定义为由空白、跳格或换行字符分隔的字符串
-m                    统计字符数。这个标志不能与 -c 标志一起使用。
-L                    打印最长行的长度。
--help                在线帮助。
--version             显示版本信息。

.. note::

	默认情况下, 显示 行数、单词数，以及字节数.

	支持多个文件统计总数.

用例
========================

计算当前文件夹下视频数(mp4文件个数)::

	ls *.mp4 | wc -l
