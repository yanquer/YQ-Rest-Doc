=======================
lsb_release
=======================

如果没有, 使用以下指令安装::

	apt install lsb-core

LSB是Linux Standard Base的缩写, Isb_release命令用来显示LSB和特定版本的相关信息(查看当前系统的发行版信息)。

语法::

	lsb_release [参数]

选项参数
=======================

-i 		显示系统名称简写
-v 		显示lsb版本信息
-d 		显示系统全称和版本号
-r 		显示版本号
-a 		显示LSB所有信息
-h 		查看lsb_release支持的所有参数

无参数时默认加上-v 参数。
