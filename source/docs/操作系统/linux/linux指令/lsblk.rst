列出所有储存装置

-d    仅列出磁盘本身，不列出分区数据
-f    同时列出磁盘内的文件系统名称
-i    使用ASCII的线段输出，不适用复杂编码
-m    同时输出该装置在 /dev 下的权限信息
-p    列出该装置的完整文件名，而不是仅列出最后的名字
-t    列出该磁盘装置的详细数据，包括磁盘队列机制、预读写的数据量大小等
