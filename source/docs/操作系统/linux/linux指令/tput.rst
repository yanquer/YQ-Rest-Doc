====================
tput
====================

功能
====================

tput 命令会利用 terminfo 数据库中的信息，来控制和更改我们的终端，比如控制光标、更改文本属性、控制屏幕，以及为文本涂色。

其中，为文本涂色的方法是:

- tput setab <color_int>：用于设置背景色
- tput setaf <color_int>：用于设置前景色
- tput sgr0：表示颜色重置

.. csv-table:: 数值定义如下
	:header: 数值, 含义

	0,		黑
	1,		红
	2,		绿
	3,		黄
	4,		蓝
	5,		洋红
	6,		黄
	7,		白
