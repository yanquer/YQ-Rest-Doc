=============================
pgrep
=============================

pgrep通过匹配其程序名, 获得正在被调度的进程的相关信息

常见选项:

-l
  同时显示进程名和PID
-o
  当匹配多个进程时，显示进程号最小的那个
-n
  当匹配多个进程时，显示进程号最大的那个

.. note::

  进程号越大，并不一定意味着进程的启动时间越晚

  pgrep查找的是程序名，不包括其参数

  pgrep相当于::

    ps –eo pid,cmd | awk ‘{print $1,$2}’ | grep KeyWord
