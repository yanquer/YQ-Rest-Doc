-n
  仅语法检查
-v
  执行前，先将内容输出（详细输出执行过程）
-x
  将使用的脚本内容显示到屏幕
-c <command>
  以非交互方式执行给定的命令, 即将内容作为脚本执行, 支持换行的文本
-e
  如果命令返回非零退出状态，则立即退出脚本
-a
  将命令添加到历史记录中，而不执行
-f
  禁用文件名扩展（通配符）
-i
  交互式运行Shell
-r
  禁用反斜杠转义
-s
  从标准输入读取命令而不从终端读取
