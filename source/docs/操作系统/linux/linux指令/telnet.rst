登录远程主机和管理(也可以简单测试ip端口是否连通).
属于应用层的协议，可用于远程登录，也可用于检测IP的端口状态

.. note::

  一般登陆成功就与目标主机端口建立链接了,
  也可以用来作为一个简单的client测试server

语法::

  telnet [host|ip [port]]
