处理两个文件之间的数据

-t    join默认以空格符分隔数据，并且比对第一个字段的数据
      如果忽略两个文件相同，则将两笔数据联成一行，且第一个字段放在第一个
-i    忽略大小写
-1    数字的1，表示第一个文件要用哪个字段来分析
-2    数字的2，表示第二个文件要用哪个字段分析
