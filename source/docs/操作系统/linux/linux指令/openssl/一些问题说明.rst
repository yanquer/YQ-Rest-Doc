为什么RSA算法要先生成私钥, 再推导公钥
==================================================

RSA说明可见: :ref:`RSA生成算法 <R_RSA生成算法>`

在openssl中,我们一般:

a. 用genrsa生成RSA私钥
#. 用rsa推导出RSA公钥

而不能:

1. 自己选择n和e生成公钥
2. 再由公钥反推私钥d

这违反了RSA算法的基本原理,破坏了其安全性。
