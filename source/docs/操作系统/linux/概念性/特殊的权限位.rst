特殊的权限位
===============================

Set-User-ID(SUID)位
  **s** 或 **S** 替换用户的 **x**
Set-Group-ID(SGID)位
  **s** 或 **S** 替换组的 **x**
粘滞位
  **t** 或 **T** 替代其他用户的 **x**
