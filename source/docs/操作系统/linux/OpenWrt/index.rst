===================================
OpenWrt
===================================

一个嵌入式 Linux, 主要实现路由功能

简单来说，OpenWRT 是一款家用路由器的操作系统，路由器安装了它才能正常的工作。
就像国内手机厂商基于开源的 Android 系统开发出各种 Rom一样。
路由器厂商也可以基于 OpenWRT 这个平台，进行二次开发，增加更多丰富的功能和有厂商特色的路由器系统。
