==================================
国际化和本地化
==================================


.. post:: 2023-02-23 23:14:15
  :tags: linux, 系统服务
  :category: 操作系统
  :author: YanQue
  :location: CD
  :language: zh-cn


程序支持国际化的行为，是通过配置环境变量 `$LANG` 来支持本地化。
语言环境的实际支持，依赖 `libc` 库提供的特性，
并要求安装 `locales` 或 `locales-all` 软件包。
`locales` 软件包需要被适当的初始化。
